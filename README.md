# VyomEdge Admin Panel

Admin dashboard for managing VyomEdge website content.

## Features

- 📝 **Blog Management** - Create, edit, delete blog posts
- 📁 **Portfolio Management** - Manage portfolio items and case studies
- ⚙️ **Services Management** - Add and update services
- 📬 **Inquiries Inbox** - View and respond to contact form submissions
- 📧 **Subscribers** - View and export newsletter subscribers

## Tech Stack

- React + Vite
- Tailwind CSS
- React Router
- Axios
- React Hot Toast

## Setup

1. Install dependencies:
```bash
npm install
```

2. Create `.env` file:
```env
VITE_API_URL=https://vyomedge-backend.onrender.com
```

3. Start development server:
```bash
npm run dev
```

4. Build for production:
```bash
npm run build
```
